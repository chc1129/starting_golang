### go test

| オプション | 効果                                                       |
| ---------- | ---------------------------------------------------------- |
| -c         | パッケージのテストのビルドのみを実行し、テストは実行しない |
| -i         | 依存するパッケージのインストールを実行する                 |
| -o         | ビルドしたテストの実行ファイルを任意のファイル名で出力する |
